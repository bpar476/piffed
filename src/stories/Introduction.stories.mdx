# Piffed Design Kit

Piffed is the portfolio design kit, a collection of React components which are useful for building pretty, organised, and effective portfolio websites. Browse the different available components
to get ideas about how you might build your portfolio.

All components are defined with `em` units which means they will scale based on the font-size of your website. For reference, these components are rendered with a base font size of 24px.

The components also accept a theme. This storybook is using a reference theme.
